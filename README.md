# Fashny

- It is an app for tourist places and vacations in egypt .
- Used Flutter with Bloc .
# API
- Rest api
# Library
- Dependance injection(geit it) : https://pub.dev/packages/get_it
- YoutubPlayer :https://pub.dev/packages/youtube_player_flutter
